# AI 影伴 v1.3 更新通知

> **发布日期**：2026-07-13
> **编译状态**：✅ BUILD SUCCESSFUL
> **APK 文件**：`历代稳定版/YinBan_v1.3.apk`（6.4 MB）

## 🎨 外观变化

| 改动 | 位置 | 说明 |
|------|------|------|
| **圆形头像** | 「我的」页面 | 可从相册选图，自动裁剪为圆形显示 |
| 患者/监护人独立头像 | 「我的」 | 两端各自设置，互不影响 |
| 监护人端底部导航 | 监护人端 | 新增「主页 / 我的」两栏，点击「我的」可查看个人信息和退出 |
| 隐私设置移除 | 患者端 → 我的 | 删除了"隐私模式"和"自动SOS"（意义不明） |
| 监护栏卡片文字修复 | 监护栏 | 4 个快捷入口的底部描述不再被截断 |
| Snackbar 通知上移 | 全局 | 从底部移到内容区上方，不再遮挡底部导航 |
| SOS 按钮文案 | 主页 | "按住 2 秒紧急求助" → "紧急求助"（点击即可触发） |
| 监护人控制栏精简 | 监护人端 | 删除无用的"视频通话"按钮，保留"消息"和"语音" |

## 🆕 功能变化

- **圆形头像**：点击头像 → 从相册选择 → 自动裁剪为圆形并保存
- **AI 小影火有记忆**：多轮对话，能记住之前聊了什么
- **聊天历史持久化**：退出再进入自动恢复（AI 和手动消息分开存储）

## 🐛 修复

- 监护栏聊天气泡和 AI 聊天历史不再串数据
- 安全守护计时器修复
- 导航栏图标居中
- 移除冗余旧代码（`HomeActivity`、`ui/home/*`）
