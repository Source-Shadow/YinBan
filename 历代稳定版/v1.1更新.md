## 🐛 Bug 修复

| 问题 | 修复方式 |
|------|----------|
| **MJPEG 视频流加载失败** | `loadDataWithBaseURL` + 内联 HTML → `loadUrl(url)` 直接加载，浏览器原生解码 MJPEG |
| **推流地址硬编码错误** | 默认推流地址改为硬件同学 IP（`192.168.137.83:5000`），支持环境变量覆盖 |

### 视频流加载对比

| | 旧方案 (v1.0) | 新方案 (v1.1) |
|---|---|---|
| 加载方式 | `loadDataWithBaseURL` + 内联 HTML | `loadUrl(url)` 直接加载 |
| HTML 代码 | 内嵌 `<img>` 标签 | 不需要，浏览器原生渲染 |
| 代码行数 | ~20 行 | 4 行 |
| 性能 | WebView 先解析 HTML 再请求图片 | 浏览器直接解码 MJPEG 流 |
